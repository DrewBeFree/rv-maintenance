# RV Maintenance

Maintenance tracker for the 2019 Thor Motor Coach Freedom Elite 26HE.

## Features

- Dashboard with vehicle info and health score ring
- 12 maintenance areas: winterize, generator, battery, roof, engine, water, HVAC, tires, and more
- Interval-based scheduling (30–365 day intervals) with overdue tracking
- Session tracking: checklist completion, notes, and date logging
- Edit mode for customizing checklist items per section
- Coming-due panel and recent activity feed

## Tech

HTML/CSS/JavaScript · Supabase SDK v2
